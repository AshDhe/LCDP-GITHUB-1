\# composants.md



\## Objectif

Application web de l'association La Clef du Parc.



\## Structure

\- /pagespubliques : pages visibles sans être connecté à son compte

\- /pagesinvite : pages visibles quand on est connecté à son compte mais qu'on n'a pas d'abonnement an cours

\- /pagesmembreactif : pages visibles quand on est connecté à son compte et qu'on a un abonnement en cours

\- /objets : éléments de code réutilisables pour intégration dans les pages

\- /worker : logique serveur Cloudflare Worker

\- /docs : documentation technique et fonctionnelle pour le développement de l'application
